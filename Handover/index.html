진행 상황 (플레이 해보는게 확인하기 편합니다.)

  • MainScene
    • Collection
      • 리스트 출력, 퍼즐 현황 등 구현
      • 퍼즐 완성 시 보상을 받는 부분 미구현
    • Puzzle
      • 퍼즐 조각 리스트 출력, 퍼즐 플레이 구현
      • 퍼즐 완성 시 보상을 받는 부분 미구현
      • 클릭 구현이 제대로 안되어서 현재 약간 미끄덩하게 클릭해야 퍼즐 조각 언락이 되고 있습니다. 수정해 주시면 될 것 같아요.
      • 퍼즐 조각 이펙트 표현 추가해 주셔야합니다.
    • Home
      • 하단 버튼 기능 구현
      • 다른 부분은 ui 만 배치된 상황입니다.
    • Store (StoreScene 포함)
      • ui 및 상품 데이터를 받아서 출력하게 구현되어 있습니다.
    • Settings
      • 토글 버튼 (sound 등 버튼) 구현과 데이터 갱신은 적용되어 있지만 실 작용은 sound만 구현되어 있습니다.’
  • PlayScene
    • 플레이 자체는 구현 완료
      • 그래도 한 번 쭉 확인 해 보시는 걸 추천합니다.
    • 난이도 조정 로직 추가해야합니다.
      • GameManager.CreateTileItemModels 메소드 안에서 로직 처리하면 됩니다.
  • Popup
    • 데일리 보너스, 피기뱅크, 골든 퍼즐 콜렉터, 인앱 번들 관련 팝업, 리뷰 팝업 미구현

주요 작업 - 데이터 관리

모든 레벨, 게임 테이블 데이터를 드라이브에 저장해 작업 후 유니티에 임포트해 사용합니다.

LevelData는 모두 다운 받아서 Assets/Resources/DB/LevelDatas 폴더에 넣으시면 됩니다. (경화님이 작업해서 올리실거에요)

GameData는 과정이 필요합니다. (exporter를 만들었어야 했는데 죄송합니다..)

  1. 변경된 데이터 테이블 시트에서 아래처럼 csv를 다운받습니다
  1. csv 파일을 열어 전체 선택해 복사 후 위의 링크에서 json으로 변경해 파일로 저장합니다.
  1. Assets/Resources/DB/GameDataTable 폴더에 넣으시면 됩니다.

+) 퍼즐 조각 데이터 만들기

  1. JigsawPuzzleGenerator 씬을 엽니다.
  1. 실행합니다.
  1. Generator 인스펙터 화면에서 퍼즐조각을 만들 퍼즐 index를 수정합니다 (PuzzleDataTable Index 입니다.)
  1. Generate를 누른 뒤 Save 버튼을 누릅니다

  • 하나의 Scene에서 프리팹으로 ui를 관리합니다.
  • ServiceLocator 방식으로 서비스(manager) 관리합니다.
  • 거의 모든 ui들은 Onsetup, UpdateUI(혹은 OnUpdateUI)로 구성되어 있습니다.
    • OnSetup 첫 실행시만 호출됩니다 (Start 메서드와 비슷)
      • 처음에만 호출하면 될 것들을 넣었습니다
      • Parameter를 받는 형식으로 만들었습니다.
    • OnUpdateUI ui가 변경되었을 경우 이것을 호출했습니다.
  • UIManager, SceneManager, PopupManager를 참고해주세요.
  • 프리팹을 ResourcePathAttribute를 사용해 인스턴스화 하고 있습니다. Popup이나 Scene 생성 시 꼭 명시 해주셔야합니다. (ResourcePathAttribute 참고)

*코드는 Game.cs 부터 확인해보시면 됩니다.
